⚠️ Si aún se ve en blanco

1. Abrir Consola (F12) → pestaña Console
   Buscar: "Uncaught ReferenceError", "Failed to load"

2. Pestaña Network → filtrar "JS"
   Verificar que estos archivos carguen (200):
   - ../styles.css
   - rental-app.css
   - ../script.js
   - ../rental-catalog.js
   - ../rental-app.js
   - React CDN (unpkg.com)

3. Si rental-catalog.js o rental-app.js dan 404:
   → Los scripts están en /public/ (un nivel arriba de /rental/)
   → Las rutas en index.html son: src="../rental-catalog.js"

4. Si React no está definido:
   → Verificar conexión a internet (CDN)
   → o cambiar a versión local de React

  4. rental-app.js → ReactDOM.createRoot(#root).render(
)
  5. Hr() → renderiza catálogo + modal
  6. rental-app.css → estilos aplicados
